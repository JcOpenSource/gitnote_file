3-5 Linux MySQL 安装


CentOS7默认安装mariadb数据库:


第一步：删除mariadb数据库 yum remove mariadb-libs.x86_64


到tmp目录： cd /tmp/


下载mysql数据 版本5.7：wget https://repo.mysql.com/mysql57-community-release-el7-8.noarch.rpm


安装mysql： yum localinstall mysql57-community-release-el7-8.noarch.rpm 


安装数据库服务：yum install mysql-community-server

（一直按enter键 有些安装步骤会跳过 需要再执行一遍）



启动服务器：service mysqld start

第一次启动过程会做一些初始化的工作


查看服务进程：ps -ef |grep mysql


[root@localhost tmp]# service mysqld start

Redirecting to /bin/systemctl start mysqld.service

3-7 Python3 开发环境




python3需要的环境依赖：yum install openssl-devel bzip2-devel expat-devel gdbm-devel readline-devel sqlite-devel mysql-devel gcc gcc-devel python-devel


python3需要的依赖


到tmp目录下载python


wget "https://www.python.org/ftp/python/3.5.0/Python-3.5.0.tgz"


使用源码编译的方式安装python


解压文件：tar -zxvf Python-3.5.0.tgz

新建一个python的地址：mkdir /usr/local/python3

先安装GCC： yum install gcc （一般都会默认安装好）

进行编译过程：Python-3.5.0/configure --prefix=/usr/local/python3

然后执行：

make

make install

这是基本的linux源码编译的过程


执行python3: /usr/local/python3/bin/python3


配置系统变量：ln -s /usr/local/python3/bin/python3 /usr/bin/python3

建立软链接

一般系统会默认搜索： /usr/bin/ 下的python环境变量

[root@localhost tmp]# python3
