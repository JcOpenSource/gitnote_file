3-5 Linux MySQL 安装


CentOS7默认安装mariadb数据库:


第一步：删除mariadb数据库 yum remove mariadb-libs.x86_64


到tmp目录： cd /tmp/


下载mysql数据 版本5.7：wget https://repo.mysql.com/mysql57-community-release-el7-8.noarch.rpm


安装mysql： yum localinstall mysql57-community-release-el7-8.noarch.rpm 


安装数据库服务：yum install mysql-community-server

（一直按enter键 有些安装步骤会跳过 需要再执行一遍）
